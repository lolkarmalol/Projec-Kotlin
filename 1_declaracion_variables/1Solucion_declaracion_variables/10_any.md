# Solucion a los enunciados de variables sin tipo

## Any

## Enunciado 1

Crea una variable llamada "valorGenerico" de tipo Any y asígnale el valor "Hola mundo".

<details>
  <summary>Solucion</summary>
  
  ```kotlin
 fun main() {
    val valorGenerico: Any = "hola mundo"
}

  ```
</details>

</details>
